# BlogStack - Blog de Dev Web, Newsletter & Intelligence Artificielle 

**Analyses hebdomadaires des dernières tendances en IA**

**BlogStack** est un blog spécialisé dans l'analyse et la curation des dernières innovations en intelligence artificielle tiré des sources officielles. Chaque vendredi, recevez une analyse approfondie des tendances qui façonnent l'avenir de l'IA via notre Newsletter.

## **À propos**

**BlogStack** est votre source de veille stratégique en intelligence artificielle. Je recherche, analyse et synthétise les avancées les plus significatives de l'IA pour vous faire gagner du temps et rester à la pointe de l'innovation.

## **Fonctionnalités**

- **Analyses hebdomadaires** - Publications privées tous les vendredis sur les tendances IA
- **Curation d'actualités** - Sélection des innovations les plus importantes
- **Newsletter Premium** - Accès anticipé et analyses exclusives par email
- **Guides approfondis** - Décryptage des technologies émergentes
- **Contenu gratuit** - Analyses publiques sur le site

## **Technologies utilisées**

- **HTML5** - Structure sémantique
- **CSS3** - Design moderne et responsive
- **JavaScript** - Interactivité
- **Font Awesome** - Icônes (via CDN)

## **Installation**

1. **Cloner le repository**
```bash
git clone https://github.com/Princejr/blogStack.git
```

2. **Naviguer dans le dossier**
```bash
cd BlogStack
```

3. **Ouvrir dans un navigateur**
- Double-cliquez sur `index.html`
- Ou utilisez un serveur local (Live Server, etc.)

## Responsive Design

Le site est entièrement responsive et optimisé pour :
- **Laptop** (1024px - 1920px)
- **Mobile** (320px - 768px)

## **Personnalisation**

### Ajouter un article
1. Créez un nouveau fichier HTML dans le dossier approprié
2. Ajoutez-le à la liste dans `blog.html`
3. Mettez à jour la navigation si nécessaire
